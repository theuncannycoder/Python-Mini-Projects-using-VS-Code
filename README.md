# Python-Mini-Projects-using-VS-Code

## Overview

This repository contains beginner-level Python mini projects developed while learning programming fundamentals. Each project focuses on building logic, understanding core concepts, and solving simple real-world problems using Python.

## Projects Included

* **bank.py** – Determines a greeting-based transaction value
* **deep.py** – Answers life-related questions using conditional logic
* **einstein.py** – Calculates energy using Einstein’s mass-energy equation
* **extensions.py** – Identifies file types based on extensions
* **faces.py** – Converts text emoticons into emojis
* **indoor.py** – Converts input text to lowercase
* **interpreter.py** – Performs basic arithmetic operations
* **meal.py** – Determines meal time based on user input
* **playback.py** – Modifies text for slower playback format
* **tip.py** – Calculates tip amount based on bill and percentage

## Skills Used

* Python basics
* Conditionals and loops
* Functions
* String manipulation
* User input handling

## How to Run

1. Download or clone this repository
2. Open the project folder
3. Run any file using:
   python filename.py
